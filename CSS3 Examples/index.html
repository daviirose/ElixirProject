Flexbox

flex-direction, row

flex-direction: row; puts your elements one after the other in a row. By default, it will not wrap your elements. That is another command.

flex-direction, column

flex-direction: column; puts your elements one after the other in a column. By default, it will not wrap your elements. That is another command.

flex-wrap, wrap

flex-wrap: wrap; will bump your elements down a line or over a line depending on your direction. This does not happen by default

Justify Content, Center

justify-content: center; will center your elements on thier line. This makes it easy to center.

Justify Content, space-around

justify-content: space-around; will give each object a even margin to seperate them.

Align Items, Center

align-items: center; will center elements verticly in container.

Flex is a container property used to orient child elements in the container in a predictable easy way. I learned how to use flexbox by going to https://css-tricks.com/snippets/css/a-guide-to-flexbox/ and reading about it and looking at the examples they showed there.

After setting the container element to dixplay: flex;, you can then use some of the flex properties shown to the left to align and position objects as shown.

Most of the properties have other funtions as well. for instance, align-items: bottom; would verticly align the iteams at the bottom of the page. These are some of the more important properties of flex.

Gradient

Gradient Up Down

Use linear-gradient(x,y) to go from x color at top to y color at bottom

Gradient, Degree/multiple colors

Put a degree value in front of your first color to set the direction of your gradient. Put in multiple color values to change from one to the other.

Gradient, Radial

Use radial-gradient to create a gradient that comes from the center to the edges.

Gradient, Percentage

Put a percentage next to your color to show how much of the gradient you want it to take up. Each gradient percentage is a percentage of the container.

Gradient, Repeating

Repeats the gradient. the size of the repeated gradient is determined by the highest percentage given.

Gradient, Transparency

Can use transparency to make a fade in like effect.

Gradient, Border

Can set a border to a gradient using border-image. make sure to set the border-image-slice to 1.

Gradients

Class follow along

Gradients are a easy and cool looking way to go from one color to the next. They can be used to create any gradient in any direction using 2 or more colors
